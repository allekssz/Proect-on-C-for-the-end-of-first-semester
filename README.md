Настоящата програма представлява опростена болнична информационна система, реализирана на програмния език C. Тя има за цел да симулира работата на регистратура в лечебно заведение, като позволява регистриране на нови пациенти, подреждането им в опашка за преглед и извикването им по ред на постъпване. Освен това програмата предоставя възможност за преглед на всички чакащи пациенти, търсене на пациент по име и извеждане на статистическа информация, като средна възраст на чакащите. Проектът демонстрира използването на структури, масиви, функции и базова валидация на входни данни.
